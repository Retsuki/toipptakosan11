# [Archtecture-World](https://toipptakosan11-71185.web.app/)

## Archtecture-World について

- 構築してきた or 知っている設計（構成図）を共有するアプリ

## なぜ共有するのか

- 共有することでジュニア層に知見を流す
- CTO やテックリードがいないスタートアップの助けになる
